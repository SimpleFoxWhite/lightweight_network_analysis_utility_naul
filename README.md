lightweight_network_analysis_utility_naul

Использование/Usage:
```
./naul --help
Options:
  -h,    --help                                  Показать эту справку
  -v,    --version                               Показать версию
  -i,    --interfaces                            Показать локальные интерфейсы
  -an,   --analyze_network (параметр)/(маска)    Полный анализ сети
  -a,    --analyze (параметр)                    Анализировать сеть
  -all,  --just_work                             Просто работать
```
